    Project

    코로날씨

    계획 -> 구현 -> 배포 -> 유지보수 까지 한 나의 첫 프로젝트이다.

    계획 - 코로나 정보와 날씨 정보, 중요 메모 기능까지

    한번에 볼수있는 앱이 필요하였다.

    구현 - 공공데이터포털의 오픈 API와 카카오 API를

    이용하여 웹 페이지를 만들었다.

    배포 - Heroku를 이용하여 웹을 배포하였다.

    검색 최적화를 하여 포털사이트들에서 검색이 되도록하였다.

    React Native를 이용하여 앱으로 만들어서 원스토어에 출시하였다.

    유지보수 - 배포하고도 계속해서 주변사람들의 의견을 받으며

    스스로 앱 과 웹을 계속 실행하면서 에러가 발생하면 수정하였다.

    그리고 불필요한 기능이 있으면 삭제하고

    필요한 기능이 생각나면 계속 추가하며 유지보수 하였다.
